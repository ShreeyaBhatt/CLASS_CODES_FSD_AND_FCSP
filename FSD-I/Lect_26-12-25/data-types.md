Datatypes:
1. **Primitive**
    - predefined or built-in Datatypes
    - store single values
    - Examples:
        * String: let name = "abc"
        * Number: let a = 5;
        * Boolean: (true/false) x == y;
        * Undefined: declared but not assigned; let a;
        * Null : o/p - Object
2. **Non-Primitive**
    - store multiple values
        * Object
            - Key:value
            - { }
        * Array
